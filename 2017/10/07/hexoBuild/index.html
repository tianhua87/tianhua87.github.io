什么是Hexo

Hexo 是一个基于 Node.js 的静态博客程序，它是一个快速、简洁且高效的博客框架。hexo 使用 Markdown解析文章，在几秒内，即可利用靓丽的主题生成静态网页。

准备工作

  • 一台CentOS7.4的主机
  • 安装git,注册github账号，配置git环境
  • 安装node.js
  • 安装npm
  • 安装hexo
